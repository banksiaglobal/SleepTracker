THIS APP - an example of using InterSystems IRIS Cloud SQL and IntegratedML to sleep analysis

This project demonstrates a variant of project development with a database deployment in InterSystems IRIS Cloud SQL and using IntegratedML. The theme of our application is user sleep quality monitoring and analysis. The client side of our application was developed using Angular (TS). The server side of our application was developed using FastAPI (Python), DB-API was used to connect to the database. The database is hosted in InterSystems IRIS Cloud SQL, and we used one of its features, IntegratedML, to predict sleep quality.

InterSystems IRIS Cloud SQL

We use the DB-API to connect to IRIS Cloud SQL from within our application. To do this, we set up a deployment in the IRIS Cloud SQL portal and write the connection data in .env file. The database connection is in the file database.py.

IntegratedML

The "sleeps" table of our database is populated with the generated data (you can find the code to auto-populate the table at generator.py). Then the machine learning algorithms provided by the cloud portal in the corresponding sections (Create, Train, Validate) are sequentially applied to the data of this table. Then a sql-query is generated in the Prediction section, which we use in our code for sleep analysis. Example of a sql-query being used: SELECT PREDICT(sleeps use sleeps) as prediction, quality, PROBABILITY(sleeps use sleeps for '3') as probability_quality, * FROM SQLUser.Sleeps where user_id = 1 and id = 1
